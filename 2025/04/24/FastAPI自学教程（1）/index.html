1. 安装 Anaconda

1.1 下载 Anaconda

推荐使用 清华镜像源 下载最新版安装包（速度快）：

  • 访问 清华大学开源软件镜像站-Anaconda
  • 选择与系统匹配的版本（如 Windows 用户选择 Anaconda3-2024.06-Windows-x86_64.exe）

1.2 安装 Anaconda

按以下关键步骤操作：

  1. 双击安装包，选择安装路径（建议非C盘，如 D:\Anaconda3）

  2. 安装类型选择 All Users

  3. 勾选以下选项：

    • ☑️ Add Anaconda to PATH（自动配置环境变量）
    • ☑️ Register Anaconda as default Python（设为默认解释器）
